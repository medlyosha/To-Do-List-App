# To-Do-List-App

A simple To-Do List application that allows you to create, mark, and delete tasks. Implemented using the MVC architecture and UIKit without Storyboard.

## Features

- Add new tasks.
- Mark tasks as completed.
- Delete tasks.

## Used Libraries

- SnapKit
- CoreData
