A simple to-do list application that allows you to create, mark, and delete tasks. Implemented using the MVC architecture and UIKit.

## Features

- Add new tasks.
- Mark tasks as completed.
- Delete tasks.

## Requirements

- iOS 13.0 and above
- Xcode 11.0 and above
